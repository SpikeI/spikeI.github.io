变量

全局变量：PHP在很多网站开发中用来作为后端的逻辑，我们通过全局变量可以直接访问HTTP协议的一些字段（应该是apache和php之间通过接口进行传递），如\$_REQUEST中就可以访问\$_GET变量和\$_POST变量，以及$_SERVER中保存着HTTP头的相关信息。

​ 弱类型：PHP中变量为弱类型，即不要求必须声明变量类型。

​ 可支持的变量种类有：

  • Integer 整数

  • Float 浮点数

  • String 字符串

  • Boolean 布尔值

  • Array 数组

  • Object 对象

操作符

恒等 “===”

在等号前后表达式的值和变量类型都相同时才返回true，否则返回false。

错误抑制操作符 “@”

警告将被抑制，若在PHP配置文件中设置了track_error属性，错误信息会被保存在$php_errormsg中。

执行操作符()

魔术方法

​ 在类中可以定义一些魔术方法，这些魔术方法会在执行特定的行为时自动调用。CTF比赛中出现过关于反序列化方面的漏洞很多都是关于魔术方法的。举一些简单的例子：

__construct()

定义这种方法后在实例化对象时会自动调用，如：

__destruct()

在类被销毁的时候会自动调用。

__tostring()

在雷伊字符串的形式输出的时候会自动调用。

__sleep()

在对象序列化的时候会自动调用

__wakeup()

在对象反序列化的时候会自动调用

关于反序列化时调用wakeup函数有一个CVE——CVE-2016-7124，如果序列化字符串中表示元素的数量比对象中的真实元素数量多那么将会跳过wakeup函数。
